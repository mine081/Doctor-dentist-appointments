# 🏥 Medical & Dental Appointment Scheduler

A modern appointment scheduling application developed with **Dart** and **Flutter** that allows users to schedule, manage, and organize appointments with doctors and dentists in a simple and intuitive way.

---

## 📋 Features

- 👨‍⚕️ Schedule medical appointments
- 🦷 Schedule dental appointments
- 📅 View all upcoming appointments
- ✏️ Edit existing appointments
- ❌ Cancel appointments
- 🔍 Search appointments
- 📱 Clean and user-friendly interface
- ⚡ Fast and responsive performance

---

## 🚀 Technologies Used

- **Dart**
- **Flutter**
- Material Design
- Object-Oriented Programming (OOP)

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/medical-dental-scheduler.git
```

Navigate to the project folder:

```bash
cd medical-dental-scheduler
```

Install dependencies:

```bash
flutter pub get
```

Run the application:

```bash
flutter run
```
